# GitHub Foundation

A Python-based project designed to provide fundamental utilities and helper functions for common programming tasks.

## Overview

GitHub Foundation is a lightweight Python project that includes basic utility functions to help developers with everyday programming needs. The project is focused on simplicity, clarity, and providing reusable code components.

## Features

- **Sum Functions**: 
  - `sum_two_numbers(a, b)` - Adds two numbers together
  - `sum_multiple_numbers(*args)` - Adds any number of arguments

- **Well-documented code** with docstrings and examples
- **Easy to use and extend**
- **Beginner-friendly** learning resource

## Getting Started

### Prerequisites
- Python 3.6 or higher

### Installation

1. Clone the repository:
```bash
git clone https://github.com/santoshbeora/github-foundation.git
cd github-foundation
```

2. Run the project:
```bash
python hello_world.py
```

## Usage Examples

### Sum Two Numbers
```python
from hello_world import sum_two_numbers

result = sum_two_numbers(5, 10)
print(result)  # Output: 15
```

### Sum Multiple Numbers
```python
from hello_world import sum_multiple_numbers

result = sum_multiple_numbers(1, 2, 3, 4, 5)
print(result)  # Output: 15
```
